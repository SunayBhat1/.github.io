Projects/Relevant Coursework

The following is a list of major academic projects done through coursework, compeitions, etc along with links to resources.

Reinforcement Learning (RL)

In this graduate course group project, we apply the latest in RL algorithmic techniques to solve a complex controls problem that is an extension of the classic cart pole problem.

Report Github Repository

Graph Theory

In a series of graduate course group projects we explored graph theory and network properties in stock market data, traffic data, scoial network data, and randomly generated Erdos-Renyi and Barabási–Albert networks. Properties such related to vertex degrees, random walk, modularity and community structure, edge weights, and more were explored to generate real-world insights. We also utilized inverse reinforcement learning to emulate intelligent behavior for an RL agent.

Autonomous Tour Guide Robot

My senior design undergraduate course project was a autonomous robot tour guide of the electrical engineering department building. We employed a full embedded system design from the PCB, sensor components, vehicle and robotic suystems, C++ coding, etc to develop the initial prototype and demo tour in one semester.

Speaker Recognition

This graduate course group project was a speaker verification algorithm that utilized speech Cepstrum coefficients, Dynamic Time Warping, and Mahalnobis Distance to determine if two audio clips are from the same speaker.
